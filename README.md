# @ianwalter/eslint-config-vue
> Ian's [ESLint][eslintUrl] configuration for [Vue.js][vueUrl]

[![npm page][npmImage]][npmUrl]

## Installation

```console
yarn add @ianwalter/eslint-config-vue --dev
```

## Usage

In `package.json`:

```js
  "eslintConfig": {
    "root": true,
    "extends": [
      "@ianwalter/eslint-config-vue"
    ]
  }
```
